# 论文精读笔记模板使用说明

## 📁 模板清单

| 模板 | 用途 | 适用场景 |
|------|------|----------|
| `论文精读_快速版.md` | 快速记录要点 | 初读、筛选论文 |
| `论文精读_详细版.md` | 深度分析 | 重要论文精读 |
| `论文对比.md` | 多论文对比 | 同期/同类工作比较 |
| `综述笔记.md` | 领域综述 | 整理某方向所有论文 |
| `复现笔记.md` | 代码复现 | 跑论文代码时记录 |

---

## 🚀 快速开始

### 1. 选择模板
```
初读筛选 → 快速版
深入研读 → 详细版
比较多篇 → 论文对比
整理领域 → 综述笔记
复现代码 → 复现笔记
```

### 2. 复制模板
```bash
# 复制模板到你的笔记目录
cp 论文精读_快速版.md ../my_notes/Attention_Is_All_You_Need.md
```

### 3. 填写内容
- 替换 `[论文标题]` 等占位符
- 删除不适用的部分
- 根据需要添加内容

## 📋 各模板使用指南

### 快速版模板
**适用**: 快速了解论文、筛选是否值得深读

**填写顺序**:
1. 基本信息 → 2. 一句话总结 → 3. 核心方法 → 4. 关键创新 → 5. 实验结果

**Tips**:
- 一句话总结控制在20字内
- 核心方法3-5句即可
- 可先填部分，后续补充

### 详细版模板
**适用**: 重要论文、需要深入理解

**填写顺序**:
1. 基本信息 + 总结
2. 研究背景与动机
3. 核心方法（边读边填）
4. 实验部分
5. 创新点 + 局限性
6. 个人笔记

**Tips**:
- 数学公式用 LaTeX 语法: `$公式$` 或 `$$公式$$`
- 伪代码用代码块
- 消融实验要记录每个组件的作用

### 对比模板
**适用**: 比较2-3篇相关工作

**使用场景**:
- 同期发表的相似工作
- 同一系列的前后论文
- 同一问题的不同解法

**Tips**:
- 对比维度要一致
- 突出各论文的差异
- 最后给出综合分析

### 综述模板
**适用**: 整理某个研究方向的全部论文

**使用场景**:
- 开题前的文献调研
- 写综述论文
- 系统学习某领域

**Tips**:
- 先建立分类体系
- 论文清单按重要性分级
- 定期更新

### 复现模板
**适用**: 跑论文代码

**记录重点**:
- 环境配置问题
- 与论文不符的地方
- 踩坑和解决方案

**Tips**:
- 边跑边记，不要事后补
- 保存关键命令和日志
- 记录论文中遗漏的细节

## ✍️ 填写建议

### 基本信息获取
```
标题/作者/年份 → 论文首页
发表信息 → Google Scholar / DBLP
代码链接 → Papers with Code
```

### 高效阅读流程
```
1. 摘要 + 结论 → 一句话总结
2. 引言 + 相关工作 → 问题背景
3. 方法部分 → 核心方法
4. 实验部分 → 实验结果
5. 讨论/局限 → 局限性分析
```

### 写作技巧
- 用自己的话总结，不要复制粘贴
- 数学公式要理解含义，不仅是抄写
- 记录疑问，方便后续查阅
- 标注重要程度: ⭐ / ⭐⭐ / ⭐⭐⭐

## 🗂️ 文件组织建议

### 按主题组织
```

├── NLP/
│   ├── Transformer_快速.md

```

```
        例: BERT_复现.md
```

## 📌 常见问题

### Q: 模板太长，需要全部填写吗？
A: 不需要。根据论文重要性和阅读目的选择填写。快速版够用就不必用详细版。

### Q: 如何处理数学公式？
A: 使用LaTeX语法。行内公式用 `$...$`，独立公式用 `$$...$$`。

### Q: 需要保存原文吗？
A: 建议保存PDF，在笔记中链接到PDF位置。

### Q: 多久更新一次综述笔记？
A: 建议每月或每季度更新，跟踪最新进展。
