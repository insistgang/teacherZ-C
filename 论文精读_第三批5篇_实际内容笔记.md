## 1️⃣ [2-10] Variational-based Segmentation of Biopores in Tomographic Images

### 基本信息
- **作者**: Benjamin Bauer, Xiaohao Cai, Stephan Peth, Katja Schladitz, Gabriele Steidl
- **机构**: Fraunhofer ITWM, University of Cambridge, University of Kassel
- **核心贡献**: 用变分方法分割土壤CT图像中的生物孔隙

### 问题描述
- **挑战**: 孔隙与土壤X射线衰减值重叠
- **难点**: 生物孔隙细长形状，含有机物残留
- **目标**: 准确分割以支持后续结构分析

### 变分模型

**能量泛函**:
$$\min_{u \in [0,1]} \sum_{(x,y,z) \in \Omega} (\tau(z) - f(x,y,z)) s(x,y,z) u(x,y,z) + \mu TV(u)$$

**深度自适应阈值**:
$$\tau(z) = c + \phi(z)$$
其中 $\phi(z)$ 是边界像素平均灰度值

**TV正则化**:
$$TV(u) = \sum_{(x,y,z)} \sqrt{(\nabla_x u)^2 + (\nabla_y u)^2 + (\nabla_z u)^2}$$

### ADMM算法
```
u^{(r+1)} = argmin_u {⟨s, u⟩ + γ/2 ||∇u - v^{(r)} + b_1^{(r)}||² + γ/2 ||u - w^{(r)} + b_2^{(r)}||²}
v^{(r+1)} = argmin_v {μ ||v||₁ + γ/2 ||∇u^{(r+1)} - v + b_1^{(r)}||²}
w^{(r+1)} = argmin_{w∈[0,1]} ||u^{(r+1)} - w + b_2^{(r)}||²
```

### 对比方法
| 方法 | 特点 |
|:---|:---|
| 指示克里金 | 基于双阈值的空间插值 |
| Otsu阈值 | 全局灰度校正后阈值 |
| 提取孔洞 | 形态学孔洞变换 |

### 结果
- 变分方法产生**最平滑的孔隙表面**
- 保持**细孔隙连通性**
- z方向平均弦长是其他方法的**2倍**

## 2️⃣ [2-11] CornerPoint3D: Look at the Nearest Corner Instead of the Center

### 基本信息
- **作者**: Ruixiao Zhang, Runwei Guan, Xiangyu Chen, Adam Prugel-Bennett, Xiaohao Cai
- **机构**: University of Southampton, HKUST, Cornell University
- **期刊**: arXiv 2025
- **核心贡献**: 预测最近角而非中心的跨域3D目标检测

### 核心问题

**中心预测的问题**:
- LiDAR只捕获物体**近侧表面**
- 中心位置**无点云数据支持**
- 跨域时容易**过拟合尺寸分布**

### 关键洞察
```
传统方法: 预测物体中心 → 猜测尺寸
本文方法: 预测最近角 → 有点云支持 → 更鲁棒
```

### 新评估指标

**APCS-BEV/APCS-ABS**:
- 评估**靠近传感器表面**的检测能力
- 对碰撞避免更实用

### 方法架构

**CornerPoint3D**:
```
点云 → 3D骨干网络 → BEV特征 → 
    → 最近角热力图 → 角点检测
    → 尺寸/旋转回归
    → 中心偏移预测（辅助）
```

**EdgeHead（精修头）**:
- 第二阶段精修
- 引导模型关注**可学习的近侧表面**

**MSGM（多尺度门控模块）**:
- 自适应特征提取
- 增强跨域泛化

### 实验结果
- 在多个跨域任务中**超越CenterPoint**
- 结合ROS数据增强达到**SOTA**
- 实现整体框检测质量与近侧表面定位精度的**平衡权衡**
