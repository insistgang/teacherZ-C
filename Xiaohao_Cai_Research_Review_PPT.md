# 变分方法 (1/5): ROF模型基础

## Rudin-Osher-Fatemi 全变分模型

**经典ROF能量泛函**：
$$\min_{u \in BV(\Omega)} TV(u) + \frac{\mu}{2}\int_\Omega (u - f)^2 dx$$

**模型特点**：
- 全变分(TV)保持边缘
- L²数据保真项去噪
- 凸优化问题，有全局最优解

---

**关键公式解析**：

| 符号 | 含义 |
|------|------|
| $TV(u)$ | 全变分，$\int_\Omega |\nabla u| dx$ |
| $f$ | 观测图像（含噪声） |
| $u$ | 恢复的图像 |
| $\mu$ | 正则化参数 |

**图表描述**：ROF去噪效果示意图，展示噪声图像→TV正则化→去噪结果

**演讲备注**：
ROF模型是图像恢复领域的基石。Xiaohao Cai的研究大量使用了这个模型，并发现了它与图像分割之间的深刻联系。

# 变分方法 (2/5): Mumford-Shah模型

## 分段常数Mumford-Shah (PCMS) 模型

**PCMS能量泛函**：
$$E_{PCMS}(\{\Omega_i, c_i\}) = \sum_{i=0}^{K-1} Per(\Omega_i) + \frac{\lambda}{2}\sum_{i=0}^{K-1}\int_{\Omega_i}(c_i - f)^2 dx$$

**模型特点**：
- 分割区域$\Omega_i$的周长惩罚
- 区域内灰度一致性约束
- **非凸问题**，存在局部最小值

---

**PCMS vs ROF 对比**：

| 特性 | PCMS模型 | ROF模型 |
|------|----------|---------|
| 目标 | 图像分割 | 图像恢复 |
| 优化性质 | 非凸 | 凸 |
| 全局最优 | 难以保证 | 保证 |
| 计算复杂度 | 高 | 低 |

**演讲备注**：
PCMS模型是图像分割的经典方法，但其非凸性导致计算困难。Xiaohao Cai的重要贡献之一就是建立了PCMS与ROF之间的联系，从而可以用凸优化方法解决分割问题。

# 变分方法 (3/5): SLaT三阶段框架

## Smoothing-Lifting-Thresholding

```
┌─────────────────────────────────────────────────────────┐
│                    SLaT 三阶段框架                       │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  输入: 退化彩色图像 f, 相位数 K                           │
│                         ↓                                │
│  Stage 1: 平滑 (Smoothing)                               │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 对每个通道求解: min E(g) = 数据项 + 平滑项 + TV项│   │
│  └─────────────────────────────────────────────────┘   │
│                         ↓                                │
│  Stage 2: 维度提升 (Lifting)                            │
│  ┌─────────────────────────────────────────────────┐   │
│  │ RGB → Lab变换, 拼接为6维向量 g* = [RGB, Lab]     │   │
│  └─────────────────────────────────────────────────┘   │
│                         ↓                                │
│  Stage 3: 阈值化 (Thresholding)                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ K-means聚类 → 最终分割 {Ω_k}                    │   │
│  └─────────────────────────────────────────────────┘   │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

- 维度提升：融合RGB和Lab双颜色空间
- 解耦设计：更改K无需重算Stage 1和2
- 退化鲁棒：专门处理噪声、模糊、信息丢失

SLaT框架是Xiaohao Cai 2015年的重要工作。它巧妙地将恢复和分割解耦，并通过维度提升增强颜色信息。这个框架至今仍被广泛引用。

# 变分方法 (4/5): T-ROF方法

## 阈值化ROF分割方法

**核心定理 (PCMS与ROF的联系)**：

> 对于二相分割，PCMS模型的部分最小化器可以通过对ROF模型的**最小化器进行阈值化**获得。

$$\text{分割} = \text{恢复} + \text{阈值化}$$

---

**T-ROF算法流程**：

```
1. 初始化: FCM聚类获得初始码本 m_i
2. 循环直到收敛:
   a. 求解ROF: u* = argmin TV(u) + (μ/2)||u-f||²
   b. 阈值化: Ω_i = {x: τ_i < u*(x) ≤ τ_{i-1}}
   c. 更新均值: m_i = mean_f(Ω_i)
   d. 更新阈值: τ_i = (m_{i-1} + m_i)/2
3. 输出分割结果
```

**图表描述**：T-ROF算法流程图，展示迭代优化过程

**演讲备注**：
T-ROF方法是一个突破性发现。它证明了图像分割可以通过图像恢复加阈值化来实现，避免了PCMS的非凸性。计算复杂度O(N)与相位数K无关，这是巨大的效率提升。

# 张量分解 (2/5): Tucker分解

## Tucker分解框架

**Tucker分解形式**：
$$\mathcal{X} \approx \mathcal{G} \times_1 U^{(1)} \times_2 U^{(2)} \times_3 U^{(3)}$$

**核心概念**：
- $\mathcal{G}$: 核心张量 (Core Tensor)
- $U^{(k)}$: 各维度的因子矩阵
- $\times_k$: 模式-k乘积

---

**Tucker分解示意**：

```
原始张量 X (n1 × n2 × n3)
         ↓
    ┌─────────┐
    │  Tucker │
    │ 分解    │
    └─────────┘
         ↓
核心张量 G (r1 × r2 × r3) + 因子矩阵 {U(1), U(2), U(3)}
```

**压缩效果**：存储从 $O(n_1 n_2 n_3)$ 降至 $O(r_1 r_2 r_3 + \sum_k n_k r_k)$

**演讲备注**：
Tucker分解是一种高阶奇异值分解，通过提取核心张量和因子矩阵来压缩数据。Xiaohao Cai在此基础上提出了高效的sketching算法。

# 张量分解 (3/5): Sketching技术

## 随机Sketching加速张量分解

**核心思想**：
$$\text{使用随机投影} \quad S: \mathbb{R}^n \to \mathbb{R}^m \quad (m \ll n)$$

**Sketching算法框架** (arXiv:2301.11598):
```
1. 构造随机Sketch矩阵 S
2. 计算Sketched张量: Y = X ×_k S
3. 对小规模张量Y执行分解
4. 恢复原规模的分解结果
```

---

**关键创新**：

| 技术 | 传统方法 | Sketching方法 |
|------|----------|---------------|
| 复杂度 | O(n^d) | O(m·n^{d-1}) |
| 精度 | 高 | 近似保持 |
| 内存 | 大 | 显著降低 |

**图表描述**：Sketching过程示意图，展示随机投影如何降维

**演讲备注**：
Sketching技术通过随机投影大幅降低计算复杂度。Xiaohao Cai的论文证明了这种方法在保持精度的同时，可以将计算复杂度降低一个数量级。

# 张量分解 (4/5): Tensor Train分解

## Tensor Train (TT) 格式

**TT分解形式** (arXiv:2308.01480):
$$\mathcal{X}(i_1, ..., i_d) = \sum_{r_0,...,r_d} G_1(i_1)_{r_0,r_1} G_2(i_2)_{r_1,r_2} ... G_d(i_d)_{r_{d-1},r_d}$$

**存储优势**：
```
原始: O(n^d)  →  TT格式: O(d·n·r²)
当 d=10, n=100, r=10:
原始: 10^20  →  TT: 10^5 (降低15个数量级!)
```

---

**TT分解的应用场景**：
- 高维量子态模拟
- 大规模机器学习
- 医学影像重建
- 不确定性量化

**图表描述**：TT分解结构图，展示核心张量的链式连接

**演讲备注**：
Tensor Train是一种特殊的张量分解格式，特别适合处理高维数据。它的存储复杂度与维度呈线性关系，使得原本不可计算的问题变得可行。

## 多尺度一致性图割(MCGC)方法 (arXiv:1903.08481)

**问题**：从TLS点云中分割单棵树木

**技术挑战**：
- 点云密度不均匀
- 树枝交叉重叠
- 地形起伏影响

---

**MCGC算法框架**：

```
TLS点云
    ↓
┌──────────────────┐
│  多尺度特征提取  │ → 不同半径的局部特征
└──────────────────┘
    ↓
┌──────────────────┐
│  图构建         │ → 节点=点, 边=邻接关系
└──────────────────┘
    ↓
┌──────────────────┐
│  一致性图割     │ → 分割单木
└──────────────────┘
```

**图表描述**：TLS点云树木分割结果可视化

**演讲备注**：
MCGC方法解决了实际林业调查中的关键问题。通过多尺度特征和一致性约束，可以准确分割出每棵树木，为森林资源调查提供技术支持。

# 深度学习融合 (2/5): 扩散模型

## 差异条件扩散模型 (arXiv:2405.04974)

**问题**：脑MRI分割中的域偏移

**核心思想**：
$$\text{分割掩码生成} = \text{扩散去噪} + \text{图像差异条件}$$

---

**Discrepancy-based Diffusion架构**：

```
输入图像 I
    ↓
┌────────────────────────────────────┐
│         图像特征提取                │
│    ┌────────────────────────┐     │
│    │  CNN编码器            │     │
│    └────────────────────────┘     │
└────────────────────────────────────┘
    ↓
┌────────────────────────────────────┐
│      Discrepancy计算               │
│  D(I) = |I - E[I]|  (与模板的差异) │
└────────────────────────────────────┘
    ↓
┌────────────────────────────────────┐
│      条件扩散模型                  │
│    ┌────────────────────────┐     │
│    │  噪声 → 去噪 → 分割掩码│     │
│    └────────────────────────┘     │
└────────────────────────────────────┘
    ↓
分割结果 S
```

**图表描述**：扩散模型去噪过程示意图

**演讲备注**：
这篇论文将扩散模型引入医学影像分割。通过图像与模板的差异作为条件，引导扩散过程生成更准确的分割掩码。这是一种创新的条件化策略。
