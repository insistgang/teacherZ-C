### 旁白文本

> 「现在，让我们深入ROF模型的数学核心。
>
> ROF模型的能量泛函包含两部分：第一项是数据保真项，确保去噪后的图像不偏离原始观测太远；第二项是全变分正则项，它鼓励图像平滑，但神奇的是——它能保持边缘。
>
> 全变分是什么？简单说，它是图像梯度绝对值的积分。想象图像是一座山，全变分就是测量山体表面的总「坡度」。
>
> 关键洞察：当边缘集Γ成为水平集的特殊形式时，MS模型可以收敛到ROF模型。这个发现统一了图像处理中两个最重要的变分模型。」

### 数学公式展示

```
ROF能量泛函:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   E_ROF(u) = ∫_Ω |∇u| dx + λ∫_Ω (u - f)² dx               │
│              ──────────     ─────────────────               │
│               全变分            数据保真项                   │
│                                                             │
│   其中：                                                    │
│   • u 是待恢复的图像                                        │
│   • f 是观测到的噪声图像                                    │
│   • λ 控制去噪强度（λ越大，越接近原图）                      │
│   • Ω 是图像区域                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘

Mumford-Shah能量泛函:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   E_MS(u, Γ) = ∫_Ω\Γ |∇u|² dx + μ|Γ| + λ∫_Ω (u - f)² dx   │
│                ───────────────   ─────                        │
│                  光滑项        边缘长度                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心定理 - MS到ROF的收敛:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   当 μ → 0 时：                                             │
│   lim_{μ→0} ||u_MS - u_ROF||_L²(Ω) = 0                     │
│                                                             │
│   且收敛速率：||u_MS - u_ROF||_L²(Ω) ≤ C · √μ              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
