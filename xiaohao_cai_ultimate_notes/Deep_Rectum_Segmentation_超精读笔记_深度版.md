## 🎯 核心创新

1. **真实-合成流水线**：R-FCN + cGAN合成
2. **在线训练策略**：逐病人增量学习
3. **多通道输入**：1通道/3通道切片
4. **CLAHE预处理**：对比度受限直方图均衡

---

## 📊 方法

### RS-FCN框架

```
CT切片 → CLAHE → R-FCN（分割）→ 后处理
                 ↑
            cGAN（合成数据）
```

### 在线训练

1. 用病人1-3训练初始模型
2. 预测病人4，加入训练集
3. 重复直到病人8
4. 病人9验证，病人10测试
