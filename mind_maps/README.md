## 4. 应用领域图

### 医学影像
- **眼底图像**: 血管分割、病变检测、糖尿病视网膜病变
- **MRI**: 快速重建、压缩感知、运动校正
- **CT**: 低剂量重建、去噪增强
- **病理图像**: 细胞分割、癌症检测

### 遥感影像
- **建筑物检测**: 违建识别、变化检测、3D重建
- **地物分类**: 土地利用、作物识别
- **目标检测**: 车辆检测、船舶检测

### 射电天文
- **图像重建**: 去卷积、CLEAN算法
- **不确定性**: 误差传播、置信估计
- **大规模数据**: 在线处理、流式成像

### 自动驾驶
- **3D检测**: 点云分割、障碍物识别
- **场景理解**: 语义分割、实例分割

### 工业检测
- **缺陷检测**: 表面缺陷、裂纹识别
- **质量控制**: 异常检测、分类筛选
