### 📝 摘要翻译

本文提出了首个从高分辨率CT扫描自动重建浮游有孔虫3D生长轨迹的端到端流水线。浮游有孔虫是具有复杂多室壳体的海洋原生生物，其房室生长轨迹提供了生物体发育和生态适应的关键洞察。然而，从成像数据自动追踪房室生长仍然是一个未被充分探索的领域，现有方法严重依赖每个房室的手动分割。本研究集成了实例分割（一种在浮游有孔虫研究中未广泛探索的计算机视觉技术）与专用的房室排序算法，从高分辨率CT扫描自动重建3D生长轨迹。我们定量和定性地评估了多种实例分割方法，每种方法针对房室的不同空间特征进行优化，并检查它们对生长顺序重建准确性的下游影响。在专家标注数据集上的实验结果验证了所提方法的有效性，为大规模、客观的形态分析开辟了道路。

## 4. 📊 背景与动机

### 4.1 动态3D重建问题

```
给定: 多时相3D数据 {S_t} for t = 1, ..., T
      其中每个S_t可能是:
      - 不完整的 (部分区域缺失)
      - 有噪声的 (扫描噪声)
      - 稀疏的 (采样点少)

目标: 重建连续的4D生长轨迹 X(x, t)
      其中 x ∈ ℝ³ 是空间坐标
            t ∈ [0, T] 是时间
```

### 4.2 传统方法及其局限

```
独立重建方法:
    对于每个时间点 t:
        1. 独立重建3D形状 S_t
        2. 对齐不同时间点

    问题:
        - 忽略时序相关性
        - 对齐误差累积
        - 无法处理拓扑变化

时序插值方法:
    1. 独立重建关键帧
    2. 插值中间帧

    问题:
        - 插值可能不物理
        - 缺少生长约束
```

### 4.3 时空变分方法

```
联合优化框架:
    重建所有时间点的形状
    同时考虑:
        - 空间平滑性 (每个时间点的3D平滑)
        - 时间连续性 (相邻时间点的变化平滑)
        - 物理合理性 (符合生长规律)
```

## 5. 💡 方法详解

### 5.1 核心数学框架

#### 5.1.1 4D能量泛函

论文定义的能量泛函：

```
E(X) = E_data(X) + αE_space(X) + βE_time(X) + γE_physics(X)
```

其中：
- **E_data(X)**: 数据一致性项
- **E_space(X)**: 空间正则化项
- **E_time(X)**: 时间正则化项
- **E_physics(X)**: 物理约束项

#### 5.1.2 数据一致性项

```
E_data(X) = Σ_{t=1}^{T} ∫_Ω ρ(||S_t(x) - X(x,t)||²) dx
```

其中：
- S_t(x): 时间t的观测数据
- X(x,t): 重建的4D轨迹
- ρ: 鲁棒惩罚函数

#### 5.1.3 空间正则化项

```
E_space(X) = Σ_{t=1}^{T} ∫_Ω ||∇_x X(x,t)||² dx
```

确保每个时间点的3D形状是光滑的。

#### 5.1.4 时间正则化项

```
E_time(X) = ∫_Ω ∫_0^T ||∂_t X(x,t)||² dt dx
```

确保生长轨迹的时间连续性。

#### 5.1.5 物理约束项

```
E_physics(X) = ∫_Ω ∫_0^T L_growth(X(x,t), ∂_t X(x,t)) dt dx
```

其中L_growth是生长模型的拉格朗日量。

### 5.2 生长模型

#### 5.2.1 扩散生长模型

```
∂_t X = D ∇²X + f(X)
```

其中：
- D: 扩散系数
- f(X): 生长源项

#### 5.2.2 弹性生长模型

```
∂_t X = ∇·σ + F
```

其中：
- σ: 应力张量
- F: 外力（如重力）

### 5.3 优化算法

#### 5.3.1 交替方向乘子法 (ADMM)

```
1. 初始化 X^0
2. For k = 0, 1, 2, ... until convergence:
    a. X子问题:
        X^{k+1} = argmin_X E(X) + (ρ/2)||X - Z^k + U^k||²

    b. Z子问题:
        Z^{k+1} = argmin_Z g(Z) + (ρ/2)||X^{k+1} - Z + U^k||²

    c. 对偶变量更新:
        U^{k+1} = U^k + X^{k+1} - Z^{k+1}

3. Return X^{k+1}
```

```python
def spacetime_discretization(data, nt, nx, ny, nz):
    """
    时空离散化
    """
    # 时间网格
    t_grid = torch.linspace(0, T, nt)

    # 空间网格
    x_grid = torch.linspace(0, Lx, nx)
    y_grid = torch.linspace(0, Ly, ny)
    z_grid = torch.linspace(0, Lz, nz)

    # 4D网格
    X, Y, Z, T = torch.meshgrid(x_grid, y_grid, z_grid, t_grid)

    return X, Y, Z, T
```

#### 5.4.2 有限差分算子

```python
def spacetime_finite_difference(X, dt, dx):
    """
    时空有限差分
    """
    # 时间导数 (前向差分)
    dX_dt = (X[:, :, :, 1:] - X[:, :, :, :-1]) / dt

    # 空间梯度 (中心差分)
    grad_X = torch.stack([
        (X[:, 2:, :, :] - X[:, :-2, :, :]) / (2 * dx),
        (X[:, :, 2:, :] - X[:, :, :-2, :]) / (2 * dx),
        (X[:, :, :, 2:] - X[:, :, :, :-2]) / (2 * dx)
    ], dim=0)

    return dX_dt, grad_X
```

### 11.2 能量计算

```python
class GrowthTrajectoryEnergy(nn.Module):
    """4D生长轨迹能量泛函"""

    def __init__(self, alpha=1.0, beta=2.0, gamma=0.5):
        super().__init__()
        self.alpha = alpha
        self.beta = beta
        self.gamma = gamma

    def forward(self, X, observations, t_grid):
        """
        参数:
            X: 4D轨迹 [B, T, H, W, D]
            observations: 观测数据 [B, T, H, W, D]
            t_grid: 时间网格
        """
        # 数据一致性
        E_data = F.mse_loss(X, observations, reduction='none')
        mask = ~torch.isnan(observations)
        E_data = E_data[mask].mean()

        # 空间正则化
        grad_space = spatial_gradient(X)
        E_space = (grad_space ** 2).mean()

        # 时间正则化
        grad_time = temporal_gradient(X, t_grid)
        E_time = (grad_time ** 2).mean()

        # 物理约束
        E_physics = self.physics_constraint(X, t_grid)

        # 总能量
        E = E_data + self.alpha * E_space + \
            self.beta * E_time + self.gamma * E_physics
        return E

    def physics_constraint(self, X, t_grid):
        """生长模型约束"""
        # 扩散生长模型: ∂_t X = D∇²X
        dX_dt = temporal_gradient(X, t_grid)
        laplacian = laplacian_3d(X)
        physics_residual = dX_dt - self.D * laplacian
        return (physics_residual ** 2).mean()
```

### 11.3 ADMM优化

```python
def admm_optimize(energy_model, observations, max_iter=100):
    """ADMM优化"""

    # 初始化
    X = observations.clone()
    X[torch.isnan(observations)] = 0  # 填充缺失值
    Z = X.clone()
    U = torch.zeros_like(X)

    rho = 1.0  # ADMM参数

    for k in range(max_iter):
        # X子问题
        X.requires_grad = True
        E = energy_model(X, observations)
        grad_X = torch.autograd.grad(E, X)[0]

        # 梯度下降
        with torch.no_grad():
            X_new = X - 0.01 * (grad_X + rho * (X - Z + U))

        # Z子问题（投影）
        Z_new = project_to_constraints(X_new + U, observations)

        # 对偶更新
        U = U + X_new - Z_new

        # 更新
        X = X_new.detach()

        # 检查收敛
        residual = (X - Z).norm().item()
        if residual < 1e-4:
            break

    return X
```
