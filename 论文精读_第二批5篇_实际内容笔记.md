## 1️⃣ [2-09] Framelet-Based Algorithm for Segmentation of Tubular Structures

### 基本信息
- **作者**: Xiaohao Cai, Raymond H. Chan, Serena Morigi, Fiorella Sgallari
- **期刊**: SSVM 2011 (LNCS 6667)
- **核心贡献**: 用Framelet方法分割管状结构（血管）

### 核心算法

**Framelet滤波器**:
```
h₀ = ¼[1, 2, 1]      # 低通
h₁ = √2/4[1, 0, -1]  # 高通（梯度）
h₂ = ¼[-1, 2, -1]    # 二阶差分
```

**迭代框架**:
```
f^(i+1/2) = U(f^(i))              # 阈值拉伸
f^(i+1) = A^T T_λ(Af^(i+1/2))    # Framelet去噪
```

### 算法流程
```
1. 初始化: f^(0) = f, 计算初始区间[α₀, β₀]
2. 迭代:
   a. 用[α_i, β_i]将图像分为三部分（低于/内/高于）
   b. 对内部区域应用Framelet去噪
   c. 更新区间[α_{i+1}, β_{i+1}]
3. 直到图像变为二值图
```

### 关键特点
- **各向异性阈值**: 只沿切向方向阈值，保护边缘
- **区间自适应**: 迭代更新边界像素值范围
- **复杂度O(n)**: 每次迭代线性复杂度
- **2D/3D通用**: 支持体积数据分割

### 实验结果
| 数据集 | 像素数 | 迭代次数 | 方法 |
|:---|:---:|:---:|:---|
| 颈动脉 | 11,284 | 6 | 各向异性更好 |
| 肾血管 | 66,049 | 8 | 细节更清晰 |

## 2️⃣ [2-20] CNN-based Hierarchical Coarse-to-Fine Segmentation of Pelvic CT

### 基本信息
- **作者**: Sharmin Sultana, Adam Robinson, Daniel Y. Song, Junghoon Lee
- **期刊**: Proc SPIE Int Soc Opt Eng 2020
- **机构**: Johns Hopkins University
- **核心贡献**: 分层粗到细的盆腔器官分割

### 方法架构

**两阶段流程**:

| 阶段 | 网络 | 功能 |
|:---:|:---|:---|
| 粗分割 | 3D U-Net | 生成ROI定位图 |
| 精分割 | U-Net + GAN | 精细器官边界 |

### 精分割网络
```
生成器G: 3D U-Net → 预测器官标签
判别器D: FCN → 判断真/假分割

损失函数:
Loss = Loss_dice + Loss_G
Loss_G = ½E[(D(G(z)) - c)²]
```

### 实验结果
| 器官 | Dice | MSD(mm) | HD(mm) |
|:---|:---:|:---:|:---:|
| 前列腺 | 0.90±0.05 | 1.56±0.37 | 5.21±1.17 |
| 膀胱 | 0.96±0.06 | 0.95±0.15 | 4.37±0.56 |
| 直肠 | 0.91±0.09 | 1.78±1.13 | 6.11±1.47 |

### 关键创新
1. **分层策略**: 先定位ROI再精细分割
2. **单类训练**: 避免多类不平衡问题
3. **GAN精修**: 判别器提供全局一致性约束
