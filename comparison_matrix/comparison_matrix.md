## 矩阵4: 方法×优势/劣势

| 方法 | 优势 | 劣势 |
|------|------|------|
| ROF模型 | 理论完备、计算高效、保边缘 | 纹理丢失、参数敏感 |
| SLaT | 稀疏优化、快速收敛 | 仅适用特定结构、扩展性有限 |
| Varifold | 几何不变性、适用于形状分析 | 计算复杂、实现困难 |
| U-Net | 端到端、小样本有效、结构简洁 | 需要标注数据、内存占用大 |
| DeepLab | 多尺度特征、空洞卷积高效 | 参数量大、推理慢 |
| PointNet | 点云直接处理、置换不变 | 局部特征捕获弱 |
| PointNet++ | 层次特征、局部+全局 | 计算量大 |
| V-Net | 3D医学专用、Dice损失 | 仅限3D、数据需求大 |
| 3D U-Net | 3D体素处理、端到端 | 内存消耗极大 |
| DGCNN | 动态图卷积、局部特征强 | 计算复杂度高 |
| Mumford-Shah | 理论基础强、分段平滑 | 求解困难、慢 |
| Chan-Vese | 无需边缘、水平集优雅 | 拓扑变化受限 |
| Level Set | 拓扑灵活、边界精确 | 需要重新初始化、慢 |
| Graph Cut | 全局最优、能量最小化 | 边界平滑性差 |
| CRF | 上下文建模强、后处理有效 | 计算密集、两阶段 |
| GAN | 生成质量高、无监督学习 | 训练不稳定、模式崩溃 |
| VAE | 潜在空间连续、生成多样 | 生成模糊 |
| Transformer | 长距离依赖、全局注意力 | 计算复杂度O(n²) |
| Diffusion | 生成质量最优、稳定训练 | 推理极慢、步数多 |

## 矩阵6: 技术路线对比

| 类别 | 代表方法 | 核心思想 | 适用场景 |
|------|----------|----------|----------|
| **变分方法** | ROF, Mumford-Shah, Chan-Vese | 能量泛函最小化 | 图像去噪、分割 |
| **水平集** | Level Set, Geodesic Active Contours | 隐式曲线演化 | 边界检测、分割 |
| **图方法** | Graph Cut, Random Walker | 离散优化 | 交互式分割 |
| **CNN** | U-Net, DeepLab, FCN | 端到端特征学习 | 通用视觉任务 |
| **点云网络** | PointNet, DGCNN, PointNet++ | 点云直接处理 | 3D场景理解 |
| **Transformer** | ViT, Swin, TransUNet | 自注意力机制 | 全局建模任务 |
| **生成模型** | GAN, VAE, Diffusion | 分布学习和采样 | 图像生成、重建 |
