## 1️⃣ [2-02] 多类分割迭代ROF阈值化

### 基本信息
- **作者**: Xiaohao Cai, Gabriele Steidl
- **期刊**: EMMCVPR 2013 (LNCS 8081)
- **核心贡献**: 用迭代ROF阈值化实现多类分割

### 核心思想
```
ROF去噪 → 阈值化 → 更新阈值 → 迭代直到收敛
```

### 关键公式

**ROF模型**:
$$\min_u TV(u) + \frac{\mu}{2}\int_\Omega(u-f)^2 dx$$

**阈值化规则**:
$$\Sigma_i = \{x \in \Omega : u(x) > \tau_i\}$$

**阈值更新**:
$$\tau_i^* = \frac{1}{2}(m_{i-1} + m_i)$$
其中 $m_i = \text{mean}_f(\Omega_i)$

### T-ROF算法
```
初始化: τ^(0) with 0 < τ_1^(0) < ... < τ_{K-1}^(0) < 1
1. 求解ROF模型得到u
2. 迭代:
   2.1 Σ_i^(k) = {x : u(x) > τ_i^(k)}
   2.2 计算m_i = mean_f(Ω_i)
   2.3 更新τ_i^(k+1) = (m_{i-1} + m_i)/2
直到收敛
```

### 与Chan-Vese的联系
- 当K=2时，T-ROF等价于CV模型
- 参数关系: **λ = μ/(2(m₁-m₀))**
- λ随类间差异自适应调整

## 2️⃣ [2-03] SLaT三阶段彩色图像分割

### 基本信息
- **作者**: Xiaohao Cai, Raymond Chan, Mila Nikolova, Tieyong Zeng
- **期刊**: J Sci Comput (2017) 72:1313–1332
- **核心贡献**: 三阶段框架处理退化彩色图像

### 三阶段框架

| 阶段 | 名称 | 功能 |
|:---:|:---|:---|
| S | Smoothing | 凸变分去噪恢复 |
| La | Lifting | RGB→Lab维度提升 |
| T | Thresholding | K-means多通道阈值 |

### Stage 1: 平滑恢复
**能量泛函**:
$$E(g_i) = \frac{\lambda}{2}\int_\Omega \omega_i \cdot \Phi(f_i, g_i)dx + \frac{\mu}{2}\|\nabla g_i\|_{L^2}^2 + \|\nabla g_i\|_{L^2}$$

数据项Φ可选:
- 高斯噪声: $\Phi = (f - Ag)^2$
- 泊松噪声: $\Phi = Ag - f\log(Ag)$

### Stage 2: 维度提升
```
RGB图像 → Lab变换 → 组合6通道向量
ḡ* = (ḡ₁, ḡ₂, ḡ₃, ḡ'₁, ḡ'₂, ḡ'₃)
```
- Lab空间提供感知均匀性
- 补充RGB高度相关的不足

### Stage 3: 多通道阈值
$$\Omega_k = \{x \in \Omega : \|ḡ^*(x) - c_k\|_2 = \min_{1\le j \le K}\|ḡ^*(x) - c_j\|_2\}$$

### 优势
- 分阶段设计：K值只在最后阶段需要
- 可并行计算：各通道独立处理
- 处理多种退化：噪声、模糊、信息丢失

## 3️⃣ [2-05] 语义比例分割 (Semantic Proportions)

### 基本信息
- **作者**: Halil Ibrahim Aysel, Xiaohao Cai, Adam Prugel-Bennett
- **期刊**: arXiv 2024
- **核心贡献**: 用语义比例代替像素级标注

### 核心思想
```
传统: 需要像素级分割掩码 (昂贵)
本文: 只需各类别比例 (廉价)
```

### SPSS架构
```
输入图像 → CNN特征提取 → GAP层 → 预测比例ρ
         ↓
    特征图Y = 分割结果
```

### 损失函数

**仅用语义比例 (SPSS)**:
$$L_{sp} = \frac{1}{|\Omega_T|}\sum_{i \in \Omega_T}\|\rho_i^* - \rho_i\|^2$$

**联合损失 (SPSS+)**:
$$L_{total} = \alpha L_{sp} + (1-\alpha)L_{sm}$$
其中 $L_{sm}$ 是BCE分割损失

### 实验结果
| 数据集 | Benchmark | SPSS | SPSS+ |
|:---|:---:|:---:|:---:|
| Aerial Dubai (Mean IoU) | 71.3 | 64.2 | **71.6** |
| ISIC | 78.4 | 73.2 | **79.1** |
| Mitochondria | 83.7 | 76.5 | **84.3** |

### 优势
- 标注成本大幅降低
- 可作为plug-and-play模块
- 对噪声比例标注鲁棒

## 4️⃣ [1-04] 测度空间分割 (PMS框架)

### 基本信息
- **作者**: Salvador Moll, Vicent Pallardó-Julià, Marcos Solera
- **期刊**: Applied Mathematics & Optimization (2024) 89:66
- **核心贡献**: 统一ROF和CV的抽象框架

### Perimeter Measure Space定义
- 测度空间(X, B, ν)
- 周界泛函Per: B → [0,∞]
- Per满足: Per(∅)=0, Per(A)=Per(X\A), 次模性
- TV(u) := ∫Per(Eₜ(u))dt (coarea公式)

### 核心定理 (Linkage)
设uλ是ROF最小化子，则:
$$\Omega = \{x : u_\lambda(x) > \frac{m_0 + m_1}{2}\}$$
是CV模型的部分最小化子，参数关系:
$$\lambda = 2\mu(m_1 - m_0)$$

### PMS的六种实例
| 类型 | 应用场景 |
|:---|:---|
| 各向同性欧氏 | 标准图像处理 |
| 各向异性欧氏 | 特定几何特征 |
| 分数周界 | 非局部处理 |
| 随机游走空间 | 图上扩散 |
| Carnot-Carathéodory | 次黎曼几何 |
| 度量图 | 网络分析 |

## 💡 关键收获

1. **ROF是核心基础**: 多种方法都以ROF去噪为起点
2. **阈值化是简单有效的分割策略**: T-ROF证明迭代阈值可逼近CV
3. **色彩空间选择重要**: SLaT使用RGB+Lab双空间
4. **弱监督可行**: SPSS证明比例信息足以训练分割模型
5. **理论统一有价值**: PMS框架揭示ROF-CV本质联系
