> 
> 问题来了：望远镜数量有限，只能在傅里叶域的某些点采样，大部分区域是空的。这就是「稀疏采样」问题。
> 
> 从稀疏的傅里叶采样恢复完整图像，这是一个病态反问题！

**图表需求**:
```

观测模型：V(u,v) = ∫∫ I(l,m) e^(-2πi(ul+vm)) dl dm

其中：
- V(u,v)：可见度函数（观测数据）
- I(l,m)：天体亮度分布（想恢复的图像）
- (u,v)：基线坐标（采样位置）

问题：V只在有限个(u,v)点有值！

      ●    ●
    ●          ●      傅里叶域
      ●    ●         （稀疏采样）
    ●      ●
```

#### 【02:00-02:30】稀疏重建的引入

**画面**:
- 压缩感知概念图
- 稀疏先验的示意

**旁白**:
> 如何解决这个病态问题？答案是：利用先验知识。
> 
> 天体图像有一个特点——稀疏性。大部分区域是黑的，只有少数亮点（恒星、星系、喷流等）。
> 
> 压缩感知理论告诉我们：如果信号在某个域中稀疏，就可以用远少于奈奎斯特采样的数据完美恢复！这正是射电天文成像的救命稻草。

### 【02:30-07:00】核心方法

#### 【02:30-03:30】稀疏重建框架

**画面**:
- 优化问题形式化
- L1范数的几何解释

**旁白**:
> 稀疏重建的核心是求解一个优化问题：
> 
> 最小化图像的稀疏度，同时保证和观测数据一致。
> 
> 用数学语言：minimize ||ΨI||₁ subject to ||AI - V||₂ ≤ ε
> 
> 这里，Ψ是稀疏变换（比如小波），A是观测矩阵（傅里叶采样），V是观测数据。

**图表需求**:
```
稀疏重建优化问题：

minimize  ||ΨI||₁  (稀疏性)
  I
subject to  ||AI - V||₂ ≤ ε  (数据一致性)

其中：
- I：待恢复的图像
- Ψ：稀疏变换（小波、曲波等）
- A：观测算子（傅里叶采样）
- V：可见度数据
- ε：噪声容忍度

L1 vs L2 正则化：

     L2范数                    L1范数
    /‾‾‾‾\                   ◇
   /       \                 / \
  /         \               /   \
 不能产生稀疏解          可以产生稀疏解
```

#### 【03:30-05:00】算法实现

**画面**:
- 迭代算法流程图
- ADMM算法示意

**旁白**:
> 怎么求解这个优化问题？经典算法有：
> 
> ISTA——迭代软阈值算法，简单但慢；
> 
> FISTA——加速版ISTA，收敛更快；
> 
> ADMM——交替方向乘子法，最常用。
> 
> ADMM的核心思想是：把原问题分解成几个子问题，交替求解。每个子问题都有闭式解，计算高效。

**图表需求**:
```
ADMM算法流程：

原问题：min ||ΨI||₁  s.t.  ||AI - V||₂ ≤ ε

引入辅助变量：min ||x||₁ + ||AI - V||²
              x=ΨI

ADMM迭代：
1. I更新：Iᵏ⁺¹ = (AᵀA + ρI)⁻¹(AᵀV + ρΨᵀ(xᵏ - uᵏ))
2. x更新：xᵏ⁺¹ = S₁/ρ(ΨIᵏ⁺¹ + uᵏ)
3. u更新：uᵏ⁺¹ = uᵏ + ΨIᵏ⁺¹ - xᵏ⁺¹

其中 S 是软阈值算子
```

**动画建议**:
- ADMM三个变量交替更新的动画
- 收敛曲线的绘制

#### 【05:00-06:00】不确定性量化

**画面**:
- 贝叶斯框架图
- 后验分布示意

**旁白**:
> Cai等人的工作有一个重要创新：不确定性量化。
> 
> 传统的稀疏重建只给出一幅图像，但不知道这幅图像有多可靠。某些区域可能恢复得很好，某些区域可能很不确定。
> 
> 用贝叶斯方法，我们可以计算后验分布，得到每个像素的置信区间。这对于科学解释非常重要！

**图表需求**:
```
不确定性量化：

贝叶斯框架：
P(I|V) ∝ P(V|I) · P(I)

其中：
- P(I|V)：后验分布（给定观测，图像的概率）
- P(V|I)：似然函数（数据拟合度）
- P(I)：先验分布（稀疏先验）

输出：
- 点估计：最可能的图像
- 不确定性：每个像素的方差/置信区间

     ┌─────────────────────────────────┐
     │  重建图像    不确定性图         │
     │  [点估计]    [方差分布]         │
     │                                 │
     │   ★★★          ▓▓░             │
     │  ★★★★★   →    ▓▓▓▓░            │
     │   ★★★          ▓▓░              │
     │                                 │
     │  确定区域      不确定区域       │
     └─────────────────────────────────┘
```

## 技术实现建议

### Python演示代码
```python
import numpy as np
from scipy import fft

def sparse_reconstruction_radio(vis_data, uv_coords, image_size, lambda_reg=0.1):
    """
    射电天文稀疏重建（简化版）
    """
    n_iter = 100
    image = np.zeros((image_size, image_size))
    
    for k in range(n_iter):
        # 正向：图像 → 可见度
        vis_pred = fft.fftshift(fft.fft2(fft.ifftshift(image)))
        vis_pred_samples = vis_pred[uv_coords[:, 0], uv_coords[:, 1]]
        
        # 数据一致性更新
        residual = vis_data - vis_pred_samples
        vis_update = np.zeros_like(vis_pred)
        vis_update[uv_coords[:, 0], uv_coords[:, 1]] = residual
        image += np.real(fft.fftshift(fft.ifft2(fft.ifftshift(vis_update))))
        
        # 稀疏正则化（软阈值）
        image = soft_threshold(image, lambda_reg)
        
        # 非负约束
        image = np.maximum(image, 0)
    
    return image

def soft_threshold(x, threshold):
    return np.sign(x) * np.maximum(np.abs(x) - threshold, 0)
```
