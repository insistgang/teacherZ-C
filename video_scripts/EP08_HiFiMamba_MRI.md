### 【00:30-02:30】问题背景

#### 【00:30-01:15】MRI成像原理

**画面**:
- 磁共振原理动画
- K空间采样示意
- 重建过程

**旁白**:
> MRI是怎么成像的？
> 
> 简单说，用强磁场让身体里的氢原子共振，然后测量它们发出的信号。这些信号经过傅里叶变换，就变成了图像。
> 
> 但问题是：要得到清晰的图像，需要在「K空间」采集足够多的数据点。数据越多，扫描时间越长。

**图表需求**:
```
MRI成像流程：

氢原子 → 磁场激发 → 接收信号 → K空间数据 → 重建 → 图像

K空间：
- 频域表示
- 需要逐行扫描
- 时间 ∝ 扫描线数

全采样：N条线 → 清晰图像（30分钟）
欠采样：N/10条线 → 模糊/伪影（3分钟）
```

#### 【01:15-02:00】加速成像的挑战

**画面**:
- 欠采样导致的伪影
- 现有加速方法的局限

**旁白**:
> 能不能少采集一些数据，再用算法补全？这就是加速MRI的核心思想。
> 
> 但欠采样会导致伪影——图像出现波纹、模糊。传统方法如并行成像、压缩感知有一定效果，但加速倍数有限。
> 
> 深度学习带来了新希望。训练神经网络，从少量数据重建高质量图像。

**图表需求**:
```
加速方法对比：

┌──────────────────────────────────────────┐
│ 方法          │ 加速倍数 │ 图像质量    │
├──────────────┼──────────┼─────────────┤
│ 全采样        │ 1×       │ 完美        │
│ 并行成像      │ 2-4×     │ 良好        │
│ 压缩感知      │ 4-8×     │ 中等        │
│ 深度学习      │ 8-16×    │ 良好-优秀   │
│ HiFi-Mamba   │ 10-20×   │ 优秀        │
└──────────────┴──────────┴─────────────┘

欠采样伪影：
- 混叠伪影（波纹）
- 模糊
- 结构失真
```

#### 【02:00-02:30】Mamba架构的引入

**画面**:
- Transformer vs Mamba对比
- 状态空间模型概念

**旁白**:
> 为什么选择Mamba？
> 
> Transformer在长序列上计算量大，O(n²)复杂度。MRI图像是高分辨率的，序列很长。
> 
> Mamba是2023年提出的状态空间模型，复杂度是O(n)，而且能捕捉长程依赖。这正是MRI重建需要的！

### 【02:30-07:00】核心方法

#### 【02:30-03:30】状态空间模型基础

**画面**:
- 状态方程的图解
- 连续到离散的转换
- Mamba的选择性机制

**旁白**:
> 首先理解状态空间模型。
> 
> 核心思想：用一组隐状态来表示系统，通过状态转移和观测方程来描述输入输出关系。
> 
> Mamba的创新是「选择性」机制：根据输入动态调整状态转移参数，让模型能够「选择性地」记住或遗忘信息。

**图表需求**:
```
状态空间模型：

连续形式：
h'(t) = Ah(t) + Bx(t)    (状态方程)
y(t) = Ch(t) + Dx(t)     (观测方程)

离散形式（适合GPU）：
h_t = Āh_{t-1} + B̄x_t
y_t = Ch_t

Mamba的选择性：
Ā, B̄, C 是输入x_t的函数（而非固定）
→ 根据内容动态调整

优势：
- O(n)复杂度（vs Transformer的O(n²)）
- 长程依赖捕捉
- 硬件友好（GPU并行）
```

#### 【05:00-06:00】频域先验的设计

**画面**:
- K空间的物理特性
- 先验约束的效果

**旁白**:
> 频域先验是HiFi-Mamba的核心创新。
> 
> K空间有几个物理特性：
> 
> 中心区域包含低频信息（整体结构），外围区域包含高频信息（细节边缘）。
> 
> 采样时，中心通常被密集采样，外围稀疏采样。
> 
> HiFi-Mamba利用这些先验，设计专门的损失函数和网络结构。

**图表需求**:
```
频域先验：

K空间特性：
┌─────────────────────────────────────┐
│     ○ ○ ○ ○ ○ ○ ○ ○               │
│   ○ ● ● ● ● ● ● ○  ← 中心密集采样   │
│   ○ ● ● ● ● ● ● ○                  │
│   ○ ● ● ● ● ● ● ○                  │
│   ○ ○ ○ ○ ○ ○ ○ ○  ← 外围稀疏采样   │
└─────────────────────────────────────┘

物理先验：
1. 能量集中：大部分能量在中心
2. 共轭对称：实数图像的K空间对称
3. 平滑性：相邻点相关

损失函数：
L = L_rec + λ₁L_dc + λ₂L_freq

- L_rec：重建损失（图像域）
- L_dc：数据一致性损失（K空间）
- L_freq：频域正则化
```
