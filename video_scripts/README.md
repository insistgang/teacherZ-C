## 旁白风格指南

### 语言风格
- 口语化、亲切感
- 避免过度学术化
- 用类比解释复杂概念
- 适度幽默

### 句式建议
```
✓ "你有没有想过..."
✓ "简单来说，..."
✓ "这听起来有点抽象，我们来..."
✓ "关键在于..."
✓ "这就是...的妙处"

✗ "本文提出了..."
✗ "该方法实现了..."
✗ "实验结果表明..."
```

### 节奏控制
- 每30秒左右一个要点
- 复杂概念分步骤讲解
- 段落之间有短暂停顿
- 适当重复关键信息
