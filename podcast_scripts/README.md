## 🎵 录音建议

### 环境要求
- 安静、无回声
- 使用高质量麦克风
- 避免背景噪音

### 语速建议
- 中等偏慢
- 关键处放慢
- 给听众思考时间

### BGM 建议
| 场景 | 推荐风格 |
|------|---------|
| 开场 | 轻柔钢琴、渐入 |

| 总结 | 温暖、回顾感 |
| 结尾 | 回到开场旋律 |
