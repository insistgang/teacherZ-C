### 1.1 tCURLoRA 扩展到视觉-语言模型

**问题定义**:
当前 tCURLoRA (Tensor CUR LoRA) 主要应用于医学影像分类任务，尚未探索其在大型视觉-语言模型 (VLM) 中的潜力。CLIP、BLIP、LLaVA 等模型参数量巨大，如何利用张量分解实现高效微调是关键问题。

**方法框架**:
```
┌─────────────────────────────────────────────────────────┐
│                  tCURLoRA-VLM 架构                       │
├─────────────────────────────────────────────────────────┤
│  输入: (图像, 文本) 对                                   │
│       ↓                                                 │
│  视觉编码器 (ViT) ←─ tCURLoRA 模块 ──→ 文本编码器       │
│       ↓                      ↓                          │
│  张量分解: Tucker + CUR → 低秩适配                      │
│       ↓                      ↓                          │
│  多模态融合层 → 输出                                     │
└─────────────────────────────────────────────────────────┘
```

**关键步骤**:
1. 将 VLM 注意力权重张量化: $\mathcal{W} \in \mathbb{R}^{L \times H \times D \times D}$
2. 应用 CUR 分解选择骨架行列: $\mathcal{W} \approx \mathcal{C} \cdot \mathcal{U} \cdot \mathcal{R}$
3. 仅训练 CUR 中间核心张量 $\mathcal{U}$
4. 设计跨模态张量对齐损失

**预期贡献**:
- 参数效率提升 10x（相比全量微调）
- 医学 VLM 专用高效微调方案
- 首个张量分解 VLM 微调框架

**可行性**: ★★★★☆ (已有 tCURLoRA 基础，VLM 架构成熟)

**问题定义**:

```
┌─────────────────────────────────────────────────────────┐
│           Variational-Deep Learning (VDL) 统一框架       │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  传统变分:           深度学习:         VDL统一:          │
│  ┌─────────┐        ┌─────────┐       ┌─────────┐       │
│  │ 能量泛函 │        │ 损失函数 │       │ 神经泛函 │       │

│  └─────────┘        └─────────┘       └─────────┘       │
│                                                          │
│  E_θ(u) = ∫_Ω R_θ(∇u) + λ D_θ(f, Au) dx                │
│                                                          │
│  - R_θ: 神经正则化器 (学习TV的广义形式)                  │
```

**预期贡献**:
