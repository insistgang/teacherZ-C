About SLaT

SLaT (Smoothing, Lifting, and Thresholding) is a three-stage image segmentation framework:

  1. Stage 1 - Smoothing: Apply ROF (Rudin-Osher-Fatemi) variational denoising to each RGB channel
  2. Stage 2 - Lifting: Convert RGB to Lab color space, creating a 6-dimensional feature vector (RGB+Lab)
  3. Stage 3 - Thresholding: Apply K-means clustering in the lifted 6D space

Reference: Cai, Chan, Nikolova, Zeng. "A Three-Stage Approach for Segmenting Degraded Color Images: Smoothing, Lifting and Thresholding." IEEE TIP 2015.

Algorithm Details

Stage 1: ROF Denoising

Solves: minu λ/2||f-u||² + ||∇u||TV
Uses Chambolle's projection algorithm for efficient computation.

Stage 2: Color Lifting

RGB → Lab conversion provides perceptually uniform color space.
6D features: [R, G, B, L, a, b] improve separability.

Stage 3: K-means

Clusters pixels in 6D feature space.
Key advantage: changing K doesn't require re-running Stages 1-2.

Key Advantages

  • Fast: Each stage can be computed independently
  • Flexible: Adjust K (number of segments) without re-running denoising
  • Robust: Lab color space improves segmentation on noisy/color-shifted images
  • Theoretical: Strong mathematical foundation from variational methods
