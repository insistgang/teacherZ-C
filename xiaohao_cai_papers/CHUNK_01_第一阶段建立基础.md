## 核心方法论提取

### 1. 深度学习架构选择决策树

```
任务类型分析:
├── 图像/空间数据 → CNN
│   ├── 分类 → ResNet/EfficientNet
│   ├── 分割 → U-Net/DeepLab
│   └── 检测 → YOLO/Faster R-CNN
├── 序列/时序数据 → RNN/Transformer
│   ├── 短序列 → LSTM/GRU
│   ├── 长序列 → Transformer
│   └── 视频 → CNN+RNN混合
└── 图像+序列混合 → CNN+Transformer
```

### 2. 图像分割方法谱系

```
分割方法演进:
传统方法 ──────────────────────────────────→ 深度学习方法
    │                                          │
    ├── 图割 (Graph Cuts)                    ├── U-Net系列
    ├── 随机游走 (Random Walker)             ├── DeepLab系列
    ├── 活动轮廓 (Active Contour)            ├── Mask R-CNN
    └── 变分法 (Mumford-Shah/ROF)           └── SAM (Segment Anything)
              │                                    │
              └──────────→ SLaT三阶段 ←────────────┘
                   (传统与深度学习的桥梁)
```

### 3. 变分法核心公式模板

```python
# 通用能量泛函设计模板
E(u) = E_data(u, f) + λ * E_reg(u)

# 数据项 (保真度)
E_data(u, f) = ∫Ω (u(x) - f(x))² dx

# 正则项 (平滑性/边缘保持)
E_reg(u) = ∫Ω |∇u(x)| dx  # 全变差
           = ∫Ω |∇u(x)|² dx  # Tikhonov
           = ∫Ω φ(|∇u(x)|) dx  # 泛函形式

# 求解: 梯度下降 / 对偶方法 / Split Bregman
```
