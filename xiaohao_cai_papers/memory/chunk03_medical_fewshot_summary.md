# CHUNK 03 中间摘要: 医学图像小样本学习

## 核心方法论

### 1. 小样本学习范式 (Few-Shot Learning)

**问题背景**: 医学图像标注数据极其稀缺，传统监督学习需要大量标注样本。

**N-way K-shot 协议**:
```
N-way: 类别数 (如5类疾病)
K-shot: 每类样本数 (如1-shot或5-shot)

训练流程:
- 支持集(Support Set): N×K个样本用于学习
- 查询集(Query Set): 用于评估模型性能
- Episode训练: 重复采样不同episode进行元学习
```

**医学应用场景**:
- 疾病分类: 正常 vs 异常 (2-way)
- 器官分割: 多器官同时标注
- 病变检测: 罕见病种识别

### 2. 非负子空间特征学习 (Non-negative Subspace)

**核心创新**: 用非负子空间基替代传统实数特征向量

**方法对比**:
| 特性 | 传统特征 | 非负子空间 |
|------|----------|------------|
| 表示形式 | 任意实数向量 | 非负基线性组合 |
| 可解释性 | 弱 | 强 (非负有物理意义) |
| 抗噪声 | 一般 | 强 (子空间降维) |
| 医学适配 | 一般 | 高 (符合医学数据特性) |

**算法流程**:
1. 支持集样本特征提取
2. 非负矩阵分解(NMF)学习子空间基
3. 查询集投影到子空间
4. 基于投影距离分类

### 3. 扩散模型病变检测 (Diffusion Model)

**核心思想**: 基于差异检测而非直接分类

**方法流程**:
```
训练阶段:
- 学习健康脑MRI的分布 (扩散模型)

测试阶段:
- 输入测试MRI
- 扩散模型重建
- 计算重建差异
- 差异大 → 病变区域
```

**优势**:
- 不直接学习病变多样性
- 利用健康样本训练 (数据充足)
- 生成质量高，可控性强

## 关键技术创新

1. **范式转移**: 从监督学习到元学习
2. **表示学习**: 非负约束增强可解释性
3. **检测策略**: 差异检测替代直接分类
4. **临床融合**: 图像+临床变量联合建模
