## 2. 直接可用论文识别

### 2.1 五星必读论文（直接用于大论文）

#### [2-11] CornerPoint3D: 3D检测新范式 (2022, IEEE TGRS)

**直接相关性**: ★★★★★

**为什么必读**：
- **核心创新**：预测最近角点而非中心 - 解决井盖定位精度问题
- **EdgeHead模块**：引导模型关注近表面 - 适用于井盖边缘检测
- **跨域验证**：专门解决跨域3D检测定位精度差

**井盖检测应用场景**：
```
传统方法：预测井盖中心 → 易受遮挡影响
CornerPoint3D：预测井盖角点 → 4个角点投票确定位置，鲁棒性强
```

**可复用组件**：
1. 角点预测头设计
2. EdgeHead边缘关注模块
3. 角点到检测框的转换算法
4. 跨域评估指标体系

---

#### [4-22] Cross-Domain LiDAR Detection (2022, Remote Sensing)

**直接相关性**: ★★★★★

**为什么必读**：
- **跨域LiDAR检测**：解决不同设备/场景下的点云检测问题
- **场景迁移问题**：与井盖检测场景高度匹配（不同道路、不同光照、不同井盖类型）
- **实用性强**：提供可部署的跨域适应方案

**井盖检测应用场景**：
```
问题：训练场景A（晴天水泥路）→ 测试场景B（雨天沥青路）
祖师爷方案：跨域特征对齐 + 域适应损失函数
```

**可复用组件**：
1. 域判别器设计
2. 特征对齐损失函数
3. 跨域数据增强策略
4. 源域-目标域采样策略

---

#### [2-25] Medical Few-Shot Learning (2021, Medical Image Analysis)

**直接相关性**: ★★★★☆

**为什么必读**：
- **小样本学习**：井盖缺陷样本稀缺，需要小样本方法
- **医学图像→工业检测**：方法直接迁移
- **元学习框架**：支持快速适应新缺陷类型

**井盖检测应用场景**：
```
问题：井盖破损、变形样本极少（缺陷样本天然稀缺）
祖师爷方案：支持集-查询集元学习 + 度量学习
```

**可复用组件**：
1. 元学习训练协议（N-way K-shot）
2. 度量学习网络设计
3. 小样本数据增强策略
4. 元训练/元测试划分方案

## 5. 小论文创新点建议

### 5.1 小论文方向1：CornerPoint-YOLO for Manhole Detection

**标题建议**：
"CornerPoint-YOLO: Robust Manhole Detection via Corner Prediction and Cross-Domain Adaptation"

**目标 Venue**：IEEE GRSL（快速发表）、Remote Sensing

**核心创新**：
1. 将CornerPoint3D的角点预测引入2D YOLO
2. 设计EdgeHead-lite轻量边缘模块
3. 跨域适应模块（不同道路场景）

**技术路线**：
```
输入图像
    ↓
YOLOv8 Backbone
    ↓
FPN Neck
    ↓
├── 角点预测头 (创新点1)
├── 边缘感知模块 (创新点2)
└── 跨域适应模块 (创新点3)
    ↓
角点聚合 → 检测框输出
```

**实验设计**：
| 数据集 | 场景 | 样本数 | 用途 |
|:---|:---|---:|:---|
| 自建井盖数据集A | 晴天水泥路 | 2000 | 训练 |
| 自建井盖数据集B | 雨天沥青路 | 500 | 跨域测试 |
| 公开道路数据集 | 多场景 | - | 泛化测试 |

**评估指标**：mAP@0.5, FPS, 跨域性能下降率

**预期贡献**：
- 首个角点预测的2D目标检测方法
- 井盖检测新SOTA
- 跨域适应在道路场景的验证
