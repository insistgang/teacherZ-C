**论文信息**:

- **年份**: 2010
- **类型**: Book Chapter (书章)
- **领域**: 变分法; 图像分割; 基础理论
- **PDF路径**: `D:/Documents/zx/xiaohao_cai_papers/[1-04] 变分法基础 Mumford-Shah与ROF.pdf`
- **优先级**: 5/5

**核心理论**:
- **Mumford-Shah泛函** (1989):
  ```
  E(u, K) = ∫|u - f|² dx + μ∫_{Ω\K}|∇u|² dx + ν·H¹(K)
  ```
  其中:
  - u: 分割后的图像
  - f: 原始图像
  - K: 分割边界
  - H¹(K): 边界K的长度

- **ROF模型** (Rudin-Osher-Fatemi, 1992):
  ```
  E_ROF(u) = ∫|∇u| dx + (λ/2)∫(u - f)² dx
  ```
  这是Mumford-Shah的分段常数简化版

**主要内容**:
1. 变分法在图像处理中的历史背景
2. Mumford-Shah泛函的数学推导
3. ROF模型的凸优化性质
4. 连续域到离散域的转换方法
5. 数值优化算法概览

- **优先级**: 5/5

**核心创新**:
- 提出了**测度空间分割**的统一框架
- 建立了ROF去噪模型与Chan-Vese分割模型的桥梁
- 推广到非局部图像分割（通过离散加权图）

**关键数学模型**:
- **Chan-Vese模型**:
  ```
  E_CV(Λ) = Per(Λ) + μ[∫_Λ (m₁ - f)² + ∫_{Ω\Λ} (m₀ - f)²]

  ```
  λ = 2μ(m₁ - m₀)

  其中u是ROF模型的解

**主要内容**:

### 3. [2-02] 多类分割迭代ROF: Iterated ROF (2017)

**论文信息**:
- **标题**: Iterated ROF for Multi-class Segmentation
- **年份**: 2017
- **期刊**: Journal of Mathematical Imaging and Vision (JMIV)
- **领域**: 变分法; 图像分割; 多类分割
- **PDF路径**: `D:/Documents/zx/xiaohao_cai_papers/[2-02] 多类分割迭代ROF Iterated ROF.pdf`
- **优先级**: 4/5

**核心方法**:
- **迭代ROF算法**: 将两类分割扩展到多类问题
- 主要思想：通过迭代应用ROF模型实现多层次分割
- 每次迭代分离出一个类别，剩余部分继续分割

**算法流程**:
```
初始化: 输入图像f, 类别数K
for k = 1 to K:
    1. 对当前图像应用ROF模型得到u
    2. 阈值化: Λ_k = {x : u(x) ≥ threshold}
    3. 提取Λ_k作为第k类
    4. 更新剩余图像: f = f - u_k (或mask掉已分割区域)
end
```

**核心算法**:
```
输入: 图像f, 初始阈值范围[α₀, β₀]
迭代:
1. 阈值化和拉伸:
   f_j^{(i+1/2)} = 0, if f_j^{(i)} ≤ α_i
   = (f_j^{(i)} - m_i)/(M_i - m_i), if α_i < f_j^{(i)} ≤ β_i
   = 1, if f_j^{(i)} > β_i

2. Framelet去噪和平滑:
   对中间部分Λ^{(i)}应用framelet变换得到f^{(i+1)}

3. 更新阈值范围[α_{i+1}, β_{i+1}]

直到: f^{(i+1)}变为二值图像
```

**应用场景**:
- 血管分割（MRA, CTA）

- 纤维结构追踪
