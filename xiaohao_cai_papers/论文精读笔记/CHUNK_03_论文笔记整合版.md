### [2-15] 3D树木分割图割 (2019)

**论文标题**: 3D Segmentation of Trees Through Flexible Multiclass Graph Cut

**核心贡献**:
- 提出灵活多类图割框架，实现单株树木级别的精确分割

```

E(L) = E_data(L) + λ_s E_smooth(L) + λ_c E_constraint(L)
```

### [2-17] 形状签名Krawtchouk矩 (2018-2019)

**论文标题**: 3D Krawtchouk Moment Invariants for Shape Signature

**研究背景**: 经典矩理论在3D的扩展

**核心内容**:
```
Krawtchouk多项式:
K_n(x; p, N) = Σ (-1)^k * C(n,k) * C(x,k) * (p/(1-p))^k

3D Krawtchouk矩:
Q_{mnp} = Σ_x Σ_y Σ_z K_m(x)K_n(y)K_p(z) f(x,y,z)

不变量构造:
1. 中心化 (消除平移)
2. 归一化 (消除缩放)
3. 旋转不变量组合
```

| 3D Legendre | 78% | 74% |

#### [2-25] 医学图像小样本学习 (2021)

**论文标题**: Medical Image Few-Shot Learning via Meta-Learning and Task Clustering

**核心挑战**: 医学图像标注成本高，样本稀缺

**方法论框架**:
```
Meta-Training Phase (元训练阶段):
├── Support Set (K-shot): 少量标注样本
├── Query Set (N-way): 查询样本
├── Feature Extractor (Embedding Net)
├── Task Clustering (任务聚类模块) ← 核心创新
└── Prototype Network (原型网络)
```

**任务聚类核心代码**:
```python
class TaskClustering(nn.Module):
    """任务聚类模块: 将相似的任务分组"""
    def __init__(self, num_clusters=5):
        super().__init__()
        self.cluster_centers = nn.Parameter(
            torch.randn(num_clusters, feature_dim)
        )

    def forward(self, task_features):
        distances = torch.cdist(task_features, self.cluster_centers)
        cluster_assignments = torch.argmin(distances, dim=1)
        return cluster_assignments
```

**实验结果 (5-way 5-shot)**:
| 方法 | ISIC | Chest X-ray | Retinal OCT | 平均 |
|:---|:---:|:---:|:---:|:---:|
| Baseline | 65.2 | 58.7 | 71.3 | 65.1 |
| MAML | 72.1 | 64.5 | 76.8 | 71.1 |
| Prototypical Networks | 74.5 | 67.2 | 78.9 | 73.5 |
| **+ Task Clustering** | **78.3** | **70.1** | **82.4** | **76.9** |

#### [2-26] 非负子空间小样本学习 (2022)

**论文标题**: Non-negative Subspace Learning for Few-Shot Image Classification

**与[2-25]的区别**: 数学理论 + 可解释性

**核心思想**:
```
子空间假设:
"同一类的样本位于某个低维子空间中"

非负约束的作用:
1. 可解释性: 非负系数 → 基于部分的表示
2. 物理意义: 图像像素值 ≥ 0
3. 稀疏性: 非负约束促进稀疏解
4. 唯一性: 非负子空间分解在适当条件下唯一
```

**NMF基础**:
```
给定数据矩阵 X ∈ ℝ^{D×N}, X ≥ 0
寻找: X ≈ WH

其中:
├── W ∈ ℝ^{D×r}, W ≥ 0 (基矩阵)
├── H ∈ ℝ^{r×N}, H ≥ 0 (系数矩阵)
└── r: 子空间维度
```

**实验结果 (5-way 1-shot)**:
| 方法 | MiniImageNet | Caltech-101 | CUB-200 | 平均 |
|:---|:---:|:---:|:---:|:---:|
| Baseline | 48.2 | 65.3 | 42.1 | 51.9 |
| Prototypical Networks | 55.7 | 72.8 | 48.5 | 59.0 |
| MAML | 57.3 | 74.2 | 50.1 | 60.5 |
| [2-25] + Task Clustering | 59.8 | 76.5 | 52.3 | 62.9 |
| **Non-negative Subspace** | **62.4** | **78.9** | **55.7** | **65.7** |

#### [2-21] 扩散模型脑MRI病变 (2022)

**论文标题**: Diffusion Model for Brain MRI Lesion Segmentation

**扩散模型基础**:
```
核心思想: 通过逐步去噪生成数据

前向过程 (采样):
x_T ~ N(0, I)  (纯噪声)
    ↓
x_{T-1} = denoise(x_T, T-1)
x_{T-2} = denoise(x_{T-1}, T-2)
    ↓
x_0 = denoise(x_1, 0)  (数据样本)

训练目标: 学习去噪网络
ε_θ(x_t, t) = x_{t-1} + sqrt(1-β_t)ε
```

**核心创新**:
1. **解剖学感知扩散**: 利用脑部解剖结构引导分割
2. **不确定性量化**: Monte Carlo Dropout估计不确定性
3. **多尺度处理**: 在多个分辨率上进行分割

#### [2-29] 中心体分割网络 (2022)

**论文标题**: CenSegNet: A Centrosome Segmentation Network for Biomedical Images

**生物学背景**:
```
中心体 (Centrosome):
├── 功能: 微管组织中心 (MTOC)、细胞分裂纺锤体极点
├── 特点: 直径约1μm，图像中仅10-50像素
├── 研究意义: 癌症诊断、药物筛选、基础研究
└── 分割挑战: 目标极小、边界模糊、低对比度
```

**CenSegNet架构**:
```
输入: 生物医学图像 (H×W×3)
    ↓
编码器: 多尺度特征提取 + 残差连接 + 注意力模块
    ↓
瓶颈层: Dilated Convolution + 多感受野融合
    ↓
解码器: 上采样与特征融合 + Skip Connection + Deep Supervision
    ↓
分割头: 1×1 Conv + Sigmoid
    ↓
输出: 中心体概率图 (H×W×1)
```

**核心创新**:
