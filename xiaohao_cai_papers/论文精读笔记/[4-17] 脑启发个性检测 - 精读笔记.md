## 🎯 研究问题

### 脑启发计算

**核心问题**: 如何从海马体结构获得启发进行个性检测

**海马体功能**:
```
海马体 (Hippocampus):
├── 记忆形成
├── 空间导航
├── 模式分离
└── 模式完成

启发:
- 模式分离 → 特征区分
- 模式完成 → 特征补全
- 记忆编码 → 表示学习
```

**脑启发神经网络**:
```
1. 结构仿生
   - 海马体CA1-CA3结构
   - 齿状回 (DG)
   - 内嗅皮层 (EC)

2. 功能仿生
   - 稀疏编码
   - 联想记忆
   - 时序处理
```

## 📊 实验结果

### 个性检测性能

| 方法 | 准确率 | 召回率 | F1分数 |
|:---:|:---:|:---:|:---:|
| **标准CNN** | 68% | 66% | 0.67 |
| **LSTM** | 70% | 68% | 0.69 |
| **Transformer** | 72% | 70% | 0.71 |
| **[4-17] Hippd** | **76%** | **74%** | **0.75** |

| 组件 | 移除后性能下降 |
|:---:|:---:|
