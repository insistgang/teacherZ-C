## 🎯 研究问题

### 颜色空间选择问题

**核心问题**: 如何利用多个颜色空间的优势进行图像分析

**常见颜色空间**:
```
1. RGB
   - 红绿蓝三通道
   - 设备相关
   - 直观但不均匀

2. HSV
   - 色调/饱和度/明度
   - 符合人眼感知
   - 颜色分割友好

3. LAB
   - 明度/绿红/蓝黄
   - 设备无关
   - 感知均匀

4. YCbCr
   - 亮度/色度
   - 压缩友好
   - 视频常用
```

**单一颜色空间局限**:
```
1. 信息不完整
   - 每个空间侧重不同
   - 丢失某些特征

2. 应用依赖
   - 不同任务需求不同
   - 难以通用

3. 鲁棒性
   - 光照变化敏感
   - 设备差异
```
