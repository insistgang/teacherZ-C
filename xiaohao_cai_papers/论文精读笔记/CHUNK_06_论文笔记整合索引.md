**关键算法**: NSGA-II非支配排序
```python
def non_dominated_sort(population, objectives):
    """
    非支配排序 (NSGA-II核心)

    将种群划分为多个Pareto前沿层
    """
    fronts = [[]]
    domination_count = [0] * len(population)
    dominated_solutions = [[] for _ in population]

    for i, p in enumerate(population):
        for j, q in enumerate(population):
            if dominates(p, q, objectives):
                dominated_solutions[i].append(j)
            elif dominates(q, p, objectives):
                domination_count[i] += 1

        if domination_count[i] == 0:
            fronts[0].append(i)

    # 构建后续前沿...
    return fronts
```

### 2.2 双层优化理论体系

```
双层优化框架 ([4-23]):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

上层问题: min_x F(x, y*(x))
           s.t. G(x, y*(x)) ≤ 0

下层问题: y*(x) = argmin_y f(x, y)
                    s.t. g(x, y) ≤ 0

求解方法:
┌─────────────────────────────────────────────────────────────────────┐
│ 1. KKT条件转化法                                                    │
│    - 将下层KKT条件作为约束                                          │
│    - 转化为单层MPEC问题                                             │
│    - 适用: 下层凸问题                                               │
├─────────────────────────────────────────────────────────────────────┤
│ 2. 隐函数梯度法                                                     │
│    - 利用隐函数定理计算 dy*/dx                                      │
│    - 上层梯度: dF/dx = ∂F/∂x + ∂F/∂y · dy*/dx                      │
│    - 使用共轭梯度法避免Hessian求逆                                  │
├─────────────────────────────────────────────────────────────────────┤
│ 3. 迭代优化法                                                       │
│    - 交替更新上层x和下层y                                           │
│    - 内循环: K步下层优化                                            │
│    - 外循环: 上层梯度更新                                           │
└─────────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
