## 🎯 研究问题

### 嵌套采样的高维挑战

**核心问题**: 如何提高嵌套采样在高维空间中的效率

**传统嵌套采样局限**:
```
1. 约束采样困难
   - 高维空间拒绝率高
   - 难以找到满足似然约束的点

2. 椭圆近似失效
   - 高维分布非椭圆
   - 活点分布复杂

3. 计算成本高
   - 需要大量似然评估
   - 高维模型计算慢
```

**近端方法优势**:
```
近端算子 (Proximal Operator):
prox_{λf}(x) = argmin_u {f(u) + (1/2λ)||u-x||²}

优势:
- 处理非光滑问题
- 高效约束优化
- 适合高维空间
```

1. **近端约束采样**
   - 使用近端算子进行约束优化
   - 梯度信息加速
   - 自适应步长

2. **高维性能提升**
   - 50维: 2.0x加速
   - 100维: 3.3x加速
   - 500维: 6.3x加速
   - 1000维: 成功采样

3. **采样效率**
   - 标准NS接受率: 5%
   - 近端NS接受率: 35%
   - 似然评估次数减少85%
