## 🎯 研究背景与动机

### 1.1 问题定义

**核心问题**: 如何同时精确分割前列腺癌放疗涉及的多个危及器官？

**盆腔器官群组**:
```
需要勾画的器官:
├── 靶区
│   └── 前列腺 (Prostate)
├── 危及器官
│   ├── 直肠 (Rectum)
│   ├── 膀胱 (Bladder)
│   └── 股骨头 (Femoral Heads)
└── 其他
    └── 小肠 (Small Bowel)
```

### 1.2 临床价值

- **剂量限制**: 每个器官有不同的剂量约束
- **计划优化**: 多器官约束同时满足
- **效率提升**: 自动勾画减少医生工作量

---

## 🔬 核心方法论

### 2.1 多任务分割框架

```
共享编码器 + 多分支解码器:
├── 共享特征提取
├── 器官特定解码器
└── 联合优化训练
```

### 2.2 器官关系建模

```
空间关系约束:
├── 直肠位于前列腺后方
├── 膀胱位于前列腺上方
├── 左右股骨头对称分布
└── 几何约束融入损失函数
```

### 2.3 级联精化策略

```
两阶段分割:
├── 第一阶段: 粗分割 (全分辨率)
├── 第二阶段: 精分割 (ROI区域)
└── 边界细节优化
```
