## 🎯 研究问题

### 无线电干涉成像的不确定性量化

**核心问题**: 如何量化无线电干涉成像中的不确定性

**无线电干涉测量原理**:
```
无线电干涉测量:
可见度函数 V(u,v) ←→ 图像亮度 I(l,m)

测量过程:
- 多个天线接收同一射电源信号
- 测量信号间的干涉 (互相关)
- 获得可见度函数样本
```

**不确定性来源**:
```
1. 采样不确定性 (UV覆盖不完整)
   - 天线阵列配置限制
   - 只能采样部分UV平面
   - 欠采样导致重建模糊

2. 噪声不确定性
   - 接收机热噪声
   - 大气噪声
   - 射频干扰

3. 系统不确定性
   - 天线响应不一致
   - 相位校准误差
   - 增益误差
```
