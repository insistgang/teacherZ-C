## 🎯 研究背景与动机

### 1.1 问题定义

**核心问题**: 如何融合多传感器数据实现精准的树木级别映射？

**单传感器局限**:
```
LiDAR: 精确3D结构，但无光谱信息
多光谱: 光谱丰富，但无3D信息
高分辨率影像: 纹理清晰，但仅2D
```

---

## 🔬 核心方法论

### 2.1 融合策略

```
中期融合 (特征级):
├── LiDAR特征提取
├── 光谱特征提取
├── 纹理特征提取
└── 注意力机制融合
```

### 2.2 树木检测

```
多传感器树木检测:
├── CHM树顶检测
├── 光谱验证 (NDVI)
└── LiDAR结构精化
```

### 2.3 属性估计

```
多源属性估计:
├── 树高 (LiDAR)
├── 树冠面积 (多源融合)
└── 树种分类 (光谱+结构)
```
