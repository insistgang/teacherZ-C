## 🎯 研究背景与动机

### 1.1 问题定义

**核心问题**: 如何自动生成准确、完整的医学影像报告？

**临床需求**:
```
报告生成挑战:
├── 放射科医生工作量大
├── 报告质量参差不齐
├── 需要专业知识
└── 时效性要求高

自动生成的价值:
├── 提高效率
├── 标准化报告
├── 辅助初级医生
└── 减少漏诊
```

### 1.2 技术挑战

```
医学报告生成难点:
├── 专业术语准确
├── 病灶定位精确
├── 关系描述正确
├── 异常检测全面
└── 正常/异常区分
```

## 🔬 核心方法论

### 2.1 整体架构

```
编码器-解码器框架:
├── 视觉编码器
│   ├── CNN提取特征
│   └── 多尺度表示
├── 文本解码器
│   ├── Transformer/LSTM
│   └── 自回归生成
└── 注意力机制
    └── 视觉-文本对齐
```

### 2.2 多任务学习

```
联合训练:
├── 标签预测 (分类)
├── 病灶定位 (检测)
└── 报告生成 (描述)
```

### 2.3 知识注入

```
医学知识融合:
├── 医学本体
├── 解剖知识
└── 常见模式
```
