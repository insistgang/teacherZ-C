## 🎯 研究背景与动机

### 1.1 问题定义

**核心问题**: 如何有效融合医学影像和临床变量进行分类？

**单模态局限**:
```
仅影像:
├── 缺乏患者背景信息
├── 无法考虑病史
└── 忽略实验室指标

仅临床:
├── 缺乏病灶视觉信息
├── 无法定位病变
└── 诊断准确性有限

需要: 影像+临床融合
```

### 1.2 融合挑战

```
融合挑战:
├── 异构数据
│   ├── 影像: 高维张量
│   └── 临床: 低维向量
├── 不同尺度
│   ├── 影像特征复杂
│   └── 临床特征简洁
└── 缺失值处理
    └── 临床数据常不完整
```

## 🔬 核心方法论

### 2.1 融合策略

```
融合层次:
├── 早期融合
│   └── 数据级拼接
├── 中期融合 ← 本文
│   └── 特征级融合
└── 晚期融合
    └── 决策级融合
```

### 2.2 网络架构

```
双分支网络:
├── 影像分支 (CNN)
│   ├── 提取视觉特征
│   └── 输出特征向量
├── 临床分支 (MLP)
│   ├── 处理结构化数据
│   └── 输出特征向量
└── 融合模块
    ├── 注意力机制
    └── 联合分类
```

### 2.3 缺失值处理

```
处理策略:
├── 均值/中位数填充
├── 学习填充网络
├── 掩码机制
└── 不确定性建模
```
