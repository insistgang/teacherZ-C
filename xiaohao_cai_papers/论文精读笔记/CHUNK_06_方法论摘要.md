## 一、神经架构搜索(NAS)方法详解

### 1.1 [4-19] Balanced NAS (TPAMI 2021) ⭐⭐⭐⭐

**核心问题**: NAS中效率与性能的平衡

**方法论框架**:
```
传统NAS问题:
├── 搜索空间大 → 计算成本高
├── 性能vs效率权衡困难
└── 评估成本过高

解决方案:
├── 多目标优化框架
│   ├── 目标1: 模型性能(准确率)
│   └── 目标2: 计算效率(FLOPs/延迟)
├── 早期性能预测
│   ├── 训练早期指标预测最终性能
│   └── 提前淘汰劣质架构
└── 搜索空间剪枝
    ├── 基于先验知识缩小搜索空间
    └── 聚焦有潜力的架构模式
```

**关键技术**:
- 多目标进化算法
- 代理模型(Surrogate Model)
- 权重共享策略
- 渐进式搜索空间细化

### 1.2 [4-20] NAS在SEI应用

**应用场景**: 特定电子情报(SEI)领域的NAS应用

## 二、多目标跟踪(MOT)方法

### 2.1 [4-21] GRASPTrack (TIP 2020) ⭐⭐⭐⭐

**全称**: Geometric Reasoning Association Segmentation Projection Tracking

**核心架构**:
```
GRASPTrack五阶段流程:

检测(Detection)
    ↓
    输出: 目标边界框 (Bounding Boxes)
    方法: 基于CNN的目标检测器

分割(Segmentation)
    ↓
    输出: 像素级掩膜 (Masks)
    方法: 实例分割网络

关联(Association)
    ↓
    输出: ID分配 (Identity Assignment)
    方法: 几何推理 + 外观特征匹配

投影(Projection)
    ↓
    输出: 3D位置 (3D Position)
    方法: 相机几何/深度估计

跟踪(Tracking)
    ↓
    输出: 时序轨迹 (Trajectories)
    方法: 卡尔曼滤波 + 轨迹管理
```

**核心创新**:
1. **几何推理关联**: 利用几何约束改进数据关联
2. **分割辅助跟踪**: 掩膜信息提升跟踪精度
3. **3D投影**: 从2D到3D的轨迹恢复

## 四、双层优化形式化

### 4.1 [4-23] 双层优化形式化 ⭐⭐⭐

**数学框架**:
```
双层优化问题:

上层问题:
    min F(x, y*(x))
    s.t. x ∈ X

下层问题:
    y*(x) = argmin f(x, y)
             s.t. y ∈ Y

其中:
- x: 上层决策变量
- y: 下层决策变量
- F: 上层目标函数
- f: 下层目标函数
```

**求解方法**:
- 基于梯度的方法
- 隐式微分
- 近似最优性条件
- 迭代优化算法

**应用场景**:
- 超参数优化
- 元学习
- 对抗训练
- 神经架构搜索

### 5.2 [4-30] 假设检验方法

**核心方法**:
```
贝叶斯假设检验:

H0: 质量值 = 0 (无信号)
H1: 质量值 ≠ 0 (有信号)

贝叶斯因子:
    BF = P(数据|H1) / P(数据|H0)

决策规则:
    BF > 阈值 → 接受H1
    BF < 阈值 → 接受H0
```

**统计量**:
- 后验概率比
- 贝叶斯因子
- 错误发现率控制

### 5.3 [4-31] 可信区间方法

**核心方法**:
```
贝叶斯可信区间:

后验分布: P(θ|数据) ∝ P(数据|θ) × P(θ)

95%可信区间:
    [θ_lower, θ_upper] 使得
    P(θ_lower ≤ θ ≤ θ_upper | 数据) = 0.95

计算方法:
├── 分位数法
├── 最高后验密度(HPD)区间
└── 自助法近似
```

### 5.4 [4-32] 峰值统计方法
