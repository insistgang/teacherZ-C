## 🎯 研究背景与动机

### 1.1 问题定义

**核心问题**: 如何从3D点云估计结构化的方向场？

**应用需求**:
```
树木结构分析:
├── 树干检测与追踪
├── 树枝分割与分类
├── 树冠形态建模
└── 生长方向分析
```

---

## 🔬 核心方法论

### 2.1 方向场估计

```
局部方向估计 (PCA):
1. 查找K近邻
2. PCA分析
3. 主方向 = 最大特征值对应的特征向量
```

### 2.2 方向场优化

```
能量函数:
E(F) = E_data(F) + λ E_smooth(F) + γ E_structure(F)
```

### 2.3 骨架提取

```
从方向场提取骨架:
1. 检测方向奇点 (分支点)
2. 沿方向追踪路径
3. 连接骨架点
```
