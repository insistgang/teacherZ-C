## 一、小样本学习协议详解

### 1.1 N-way K-shot 核心设定

```
医学图像小样本学习协议

N-way:  类别数量 (如5类疾病)
K-shot: 每类样本数 (如1-shot或5-shot)

典型配置:
- 5-way 1-shot: 5类疾病，每类1个样本
- 5-way 5-shot: 5类疾病，每类5个样本
```

### 1.2 Episode训练范式

```
┌─────────────────────────────────────────────────────┐
│                   Episode 结构                       │
├─────────────────────────────────────────────────────┤
│  支持集 (Support Set)        查询集 (Query Set)      │
│  ┌─────────────────┐         ┌─────────────────┐   │
│  │ 类别A: K个样本   │         │ 类别A: M个样本   │   │
│  │ 类别B: K个样本   │  ───→   │ 类别B: M个样本   │   │
│  │ 类别C: K个样本   │  评估   │ 类别C: M个样本   │   │
│  │ ...             │         │ ...             │   │
│  └─────────────────┘         └─────────────────┘   │
│        ↓                          ↓                 │
│   学习类别原型/子空间         计算分类准确率          │
└─────────────────────────────────────────────────────┘
```

### 1.3 医学场景适配

| 应用场景 | N-way设定 | K-shot设定 | 挑战 |
|---------|-----------|-----------|------|
| 疾病分类 | 2-way (正常/异常) | 1-5 shot | 病变多样性 |
| 器官分割 | 多way (多器官) | 5-10 shot | 器官形态差异 |
| 病变检测 | N-way (N种病变) | 1-shot | 罕见病种 |

**来源论文**: [2-25] 医学图像小样本学习 (Medical Image Analysis 2021)

## 二、非负子空间学习方法

### 2.1 核心思想对比

```
传统特征学习                    非负子空间学习
─────────────────────────────────────────────────────
特征 = 任意实数向量              特征 = 非负子空间基的线性组合
    ↓                                ↓
可解释性弱                        可解释性强
物理意义不清晰                     符合医学数据物理意义
抗噪声能力一般                     抗噪声能力强
```

### 2.2 算法流程

```python
# 非负子空间学习算法 (来自[2-26])
def non_negative_subspace_learning(support_features, n_components):
    """
    步骤:
    1. 支持集样本学习子空间基 (非负约束)
    2. 查询集投影到子空间
    3. 基于投影距离分类
    """
    # 非负矩阵分解 (NMF)
    from sklearn.decomposition import NMF
    model = NMF(n_components=n_components, init='random')
    W = model.fit_transform(support_features.T)  # W: 子空间基
    H = model.components_

    return W  # 非负子空间基
```

### 2.3 非负约束优势

| 优势 | 说明 | 医学意义 |
|------|------|---------|
| 可解释性 | 非负权重 = 部件贡献 | 病变区域可定位 |
| 物理意义 | 像素强度非负 | 符合成像原理 |
| 稀疏性 | 自然产生稀疏表示 | 关注关键区域 |
| 抗噪性 | 子空间降维 | 减少噪声干扰 |

**来源论文**: [2-26] 非负子空间小样本学习 (IEEE TMI 2022)

    """
    N-way K-shot 学习协议

    # 支持集: N类 × K样本
    # 查询集: N类 × M样本 (评估)

    # 方法A: 原型网络
