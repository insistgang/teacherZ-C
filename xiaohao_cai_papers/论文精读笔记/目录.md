### 01. 变分法基础 Mumford-Shah与ROF

**方法论指纹**:
- 问题: 图像分割的数学基础建立
- 假设: 图像由分段光滑函数表示
- 路线: 变分法 + 能量泛函
- 验证: 理论分析

**核心公式**:
```
E(u) = ∫(u-f)² + λ∫|∇u|
```

**可复用组件**: 能量泛函设计模板、变分推导流程

---

### 02. 凸优化分割 Convex Mumford-Shah

**方法论指纹**:
- 问题: 非凸优化难题
- 假设: 凸松弛可获得全局最优
- 路线: Split Bregman迭代
- 验证: 初始化独立性验证

**核心算法**:
```
1. 凸松弛: u ∈ {0,1} → u ∈ [0,1]
2. Split Bregman迭代
3. 阈值处理
```

**可复用组件**: 凸优化分割模板、原始-对偶算法

### 06. SLaT三阶段分割

**方法论指纹**:
- 问题: 退化图像分割

- 路线: Smoothing-Lifting-Thresholding
- 验证: 退化图像数据集

**核心算法**:
```
1. Smoothing: 多尺度高斯滤波

```
