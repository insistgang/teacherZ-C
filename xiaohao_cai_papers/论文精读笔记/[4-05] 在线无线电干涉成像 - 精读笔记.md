## 🎯 研究问题

### 在线成像挑战

**核心问题**: 如何实时处理流式无线电干涉数据

**传统批处理 vs 在线处理**:
```
批处理 (Batch):
├── 收集所有数据
├── 一次性重建
├── 计算复杂度高
└── 延迟大

在线处理 (Online):
├── 数据流式到达
├── 增量更新重建
├── 计算效率高
└── 实时响应
```

**技术挑战**:
```
1. 数据流式到达
   - 观测持续进行
   - 数据量不断增长
   - 无法存储全部数据

2. 实时性要求
   - 需要快速响应
   - 延迟敏感应用
   - 动态源监测

3. 增量更新
   - 避免从头重建
   - 高效利用已有结果
   - 保持数值稳定性
```
