## 🎯 研究问题

### 离线vs在线重建对比

**核心问题**: 系统比较离线和在线重建算法的性能

**离线重建**:
```
优点:
├── 全局最优
├── 精度高
└── 充分利用所有数据

缺点:
├── 计算时间长
├── 内存需求大
├── 无法实时响应
```

**在线重建**:
```
优点:
├── 实时响应
├── 内存效率高
├── 适合流式数据

缺点:
├── 局部最优
├── 精度略低
└── 依赖初始值
```
