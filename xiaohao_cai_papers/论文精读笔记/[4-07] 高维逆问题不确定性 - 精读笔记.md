## 🎯 研究问题

### 高维逆问题的不确定性量化

**核心问题**: 如何在高维空间中进行贝叶斯不确定性量化

**逆问题定义**:
```
观测模型:
y = A(x) + ε

其中:
- y: 观测数据 (低维)
- x: 未知参数 (高维)
- A: 前向模型 (通常不可逆)
- ε: 噪声

逆问题: 给定y，估计x
```

**高维挑战**:
```
1. 维度灾难
   - 后验分布在高维空间
   - 采样困难
   - 计算复杂度高

2. 多模态
   - 后验可能有多个峰值
   - 传统方法易陷入局部最优

3. 证据计算
   - 边缘似然积分困难
   - 模型选择需要证据
```
