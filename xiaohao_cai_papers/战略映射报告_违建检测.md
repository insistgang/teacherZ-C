## 2. 直接可用论文识别

### 2.1 五星必读论文（违建检测核心）

#### [4-29] 遥感图像舰船匹配 (2021, IEEE GRSL)

**直接相关性**: ★★★★★

**为什么必读**：
- **遥感图像匹配技术**：直接可用于建筑物匹配
- **线特征提取**：建筑物轮廓提取的基石
- **GRSL级别**：适合快速发表的遥感方法

**违建检测应用场景**：
```
遥感图像舰船匹配 → 遥感图像建筑匹配
  → 检测新增/变化的建筑物
  → 提取建筑物轮廓特征
  → 时序匹配对比
```

**可复用组件**：
1. 线特征提取算法
2. 遥感图像配准方法
3. 特征匹配策略

---

#### [2-15] 3D树木分割图割 (2017)

**直接相关性**: ★★★★☆

**为什么必读**：
- **3D点云分割**：无人机LiDAR点云处理
- **图割优化**：全局最优分割方法
- **多类分割**：区分建筑/植被/地面

**违建检测应用**：
```
3D点云 → 3D树木分割方法 → 3D建筑分割
  → 从LiDAR点云提取建筑物
  → 建立数字表面模型(DSM)
  → 检测新增建筑物
```

---

**违建检测应用**：
```
粗分割 → 边界精化 → 精确建筑轮廓

```

## 4. 学术空白识别

### 4.1 祖师爷未涉足的违建检测蓝海

#### 空白1：时序违建检测系统

**现状**：Xiaohao Cai未涉及违建的时序检测

**机会**：
- 双时相/多时相遥感图像分析
- 建筑物演化轨迹追踪
- 违建发展阶段识别
- 智能巡检路线规划

**切入方式**：
```
遥感舰船匹配 ([4-29]) + 时序分析 + 违建规则
  = 首个系统性时序违建检测框架
```

---

#### 空白2：违建边界精确定位

**现状**：[2-16] 精化树木边界，未应用于建筑

**机会**：
- 建筑边界亚像素级定位
- 越界面积精确计算
- 违建严重程度评估
- 执法取证支持

---

#### 空白3：3D违建建模与分析

**现状**：3D分析针对树木，未针对建筑

**机会**：
- 违建3D形状建模
- 违建高度/体积估算
- 景观影响评估
- 3D可视化管理

---

#### 空白4：多源违建数据融合

**现状**：[2-19] 融合光学+LiDAR，但未用于违建

**机会**：
- 卫星 + 无人机 + 地面多尺度检测
- 多时相数据融合
- 提高鲁棒性和准确性

---
